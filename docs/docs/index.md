# Zentra Core Documentation

Welcome to the official documentation for **Zentra Core**, your personal, offline AI assistant.

Zentra Core is designed to be a private, modular, and powerful companion that runs entirely on your hardware.

## What is Zentra Core?

Zentra Core combines local LLMs (Large Language Models), voice interaction, system automation, and a modular architecture to provide a fully customizable AI experience without cloud dependencies.
