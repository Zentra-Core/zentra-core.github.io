FEATURES

🧠 Local AI Brain

Zentra Core supports models from Ollama and KoboldCPP. Whether you want a small fast model for quick tasks or a large, high-reasoning model for deep analysis, the choice is yours. 100% execution on your GPU/CPU.

⚡ Modular Action Engine

The system generated structured commands (JSON) that are executed by specialized plugins. This ensures reliability and allows for easy expansion.

🤖 Autonomous Agentic Loop

Watch Zentra think in real-time. By utilizing multiple steps of reasoning and dynamically selecting tools, the AI can accomplish complex, multi-stage missions without giving up.

🛡️ Zentra Code Sandbox

A native, Docker-less isolated execution environment. The AI can write and run algorithmic Python code to answer analytical questions safely.

💾 Semantic Memory

Utilizes SQLite to store not just chat history, but context and learned facts about the user. The AI evolves with you.

🎙️ Voice Interaction

Low latency Speech-to-Text and high-quality local Text-to-Speech (Piper) enable a natural, hands-free experience.

🖥️ System Bridge

Control windows, files, and applications. Zentra can act as your personal OS shell enhanced with AI.
