FEATURES

🧠 Neural Architecture

Zentra Core acts as a technical bridge for Ollama and KoboldCPP backends. It manages the inference lifecycle, ensuring that even large-scale reasoning models run efficiently on local consumer hardware without cloud leakage.

⚡ Structured Logic Engine

Instead of unpredictable text replies, our core engine generates validated JSON action tokens. These tokens interface directly with Python-based plugins, providing a deterministic bridge between AI thought and system execution.

🤖 Autonomous Agentic Loop

Watch Zentra think in real-time. By utilizing multiple steps of reasoning and dynamically selecting tools, the AI can accomplish complex, multi-stage missions without giving up.

🛡️ AST-Security Jail

A sophisticated Abstract Syntax Tree analysis layer filters all AI-generated code. This creates a secure "Jail" where Python can solve analytical problems without compromising the host operating system's integrity.

💾 Semantic Memory

Utilizes SQLite to store not just chat history, but context and learned facts about the user. The AI evolves with you.

🎙️ Voice Interaction

Low latency Speech-to-Text and high-quality local Text-to-Speech (Piper) enable a natural, hands-free experience.

🖥️ System Bridge

Control windows, files, and applications. Zentra can act as your personal OS shell enhanced with AI.
