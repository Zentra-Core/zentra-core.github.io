===================================================
 GUIDA ALL'INSTALLAZIONE DI ZENTRA CORE v0.17.0
          (Per PC "vergine" - da zero)
===================================================

1. INSTALLARE PYTHON
   - Vai su https://www.python.org/downloads/
   - Scarica Python 3.10 o superiore (consigliato 3.11/3.12)
   - Durante l'installazione, spunta "Add Python to PATH"

2. INSTALLARE OLLAMA (CONSIGLIATO)
   - Scarica da https://ollama.com
   - Esegui: ollama pull gemma2

3. SCARICARE ZENTRA CORE
   - Estrai la repository in una cartella (es. C:\ZentraCore)

4. INSTALLARE DIPENDENZE E CERTIFICATI
   - Apri il terminale nella cartella di Zentra.
   - Esegui: pip install -r requirements.txt
   - Esegui: python scripts/setup_pki.py (per abilitare HTTPS/SSL locale)

5. PRIMO AVVIO
   - Esegui: python main.py
   - Accedi alla WebUI: http://localhost:7070
   - Login default: admin / zentra

6. MCP & TOOL HUB
   - Vai in Configurazione -> MCP Bridge per installare nuovi tool da Smithery o GitHub.
