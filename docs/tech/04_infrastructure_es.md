# 🔗 4. Infraestructura Clave

La infraestructura de Zentra se basa en estándares modernos para garantizar la interoperabilidad.

- **LiteLLM**: Proporciona una interfaz unificada para comunicarse con más de 100 proveedores de modelos de IA.
- **Flask y Socket.io**: Gestionan la comunicación bidireccional en tiempo real entre el backend y la WebUI.
- **Loguru**: Sistema de registro avanzado para el monitoreo y la depuración.
- **Pydantic**: Validación rigurosa de los esquemas de configuración y los datos en tránsito.
