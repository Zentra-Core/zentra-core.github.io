# ✨ Bienvenidos a Zentra Core

Zentra Core es una interfaz neural avanzada diseñada para transformar tu computadora en un compañero inteligente capaz de escuchar, hablar y actuar.

## 👤 ¿Qué es Zentra?
Piensa en Zentra como el "cerebro" de tu computadora. No es solo un simple programa de chat, sino una entidad digital que puede adoptar diferentes **Almas (Personalidades)** para ayudarte con tus tareas diarias, desde la gestión de archivos hasta la conversación creativa.

## 🗣️ Interacción Natural: Habla y Escucha
Zentra está diseñado para un uso natural, tal como hablarías con una persona:
- **Te Escucha**: Usando el sistema "Push-to-Talk", puedes hablar con Zentra. El sistema transformará instantáneamente tu voz en texto.
- **Te Responde**: Zentra no solo escribe mensajes; tiene su propia voz. Cada respuesta se lee en voz alta utilizando una síntesis de voz clara y natural.

## 🎭 Personalidades (Almas)
Zentra no tiene un carácter fijo. Puedes cargar diferentes "Almas" para cambiar la forma en que interactúa. ¿Quieres un asistente serio y profesional? ¿O tal vez una IA con un carácter más informal y amigable? Puedes cambiarlo en cualquier momento con un simple clic.

## 🖥️ Dos Caras de la Misma Moneda: Consola y WebUI
Zentra es un sistema moderno que ofrece dos formas de interactuar:
1. **Consola Zentra (El Corazón)**: Esta es la ventana de comandos que ves al iniciar el programa. Es donde el sistema "piensa", realiza comprobaciones de seguridad y te permite gestionarlo todo rápidamente con el teclado.
2. **WebUI (La Cara)**: Esta es la interfaz moderna que abres en tu navegador. Aquí puedes chatear visualmente, enviar imágenes y configurar cada detalle con una interfaz gráfica sencilla e intuitiva.

## 🔒 Privacidad e Inteligencia Local
A diferencia de muchos asistentes populares, Zentra Core prioriza tu privacidad. Muchas de sus funciones están diseñadas para ejecutarse directamente en tu computadora (**localmente**). Esto significa que tus datos y conversaciones permanecen contigo, sin necesidad de viajar por Internet.
