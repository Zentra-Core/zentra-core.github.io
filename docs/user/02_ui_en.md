# 🖥 2. User Interface (Safe Scrolling UI)

Zentra's terminal interface uses an anchored architecture:
- **Dashboard (Top):** Live hardware stats (CPU, RAM, VRAM).
- **Status Bar:** Shows current state (READY, THINKING, SPEAKING), active model, and personality.
- **Chat Area:** Interaction history scrolls from row 7 downwards, keeping the dashboard visible at all times.
- **Neural Link:** An optional graphical mini-panel for quick media access.
