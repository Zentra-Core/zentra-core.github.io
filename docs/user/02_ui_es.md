# 🖥 2. Interfaz de Usuario (TUI)

La interfaz de terminal de Zentra utiliza una arquitectura anclada para maximizar la visibilidad:
- **Dashboard (Superior):** Estadísticas de hardware en tiempo real (CPU, RAM, VRAM).
- **Barra de Estado:** Indica el estado actual (LISTO, PENSANDO, HABLANDO), el modelo de IA activo y la personalidad cargada.
- **Área de Chat:** El historial de interacción se desplaza desde la fila 7 hacia abajo, manteniendo el panel de control siempre visible.
- **Neural Link:** Un mini-panel gráfico opcional para acceso rápido a medios.
