# 🖥 2. Interfaccia Utente (TUI)

L'interfaccia terminale di Zentra utilizza un'architettura ancorata per massimizzare la visibilità:
- **Dashboard (In Alto):** Mostra le statistiche hardware in tempo reale (CPU, RAM, VRAM).
- **Barra di Stato:** Indica lo stato corrente (PRONTO, PENSANDO, PARLANDO), il modello IA attivo e la personalità caricata.
- **Area Chat:** La cronologia dell'interazione scorre dalla riga 7 in giù, mantenendo la dashboard sempre visibile.
- **Neural Link:** Un mini-pannello grafico opzionale per l'accesso rapido ai media.
