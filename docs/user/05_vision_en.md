# 👁️ 5. Vision & Multimodal Interaction

Zentra Core natively supports computer vision through the `VISION_AI` and `WEBCAM` plugins.

- **Image Analysis**: Drag an image into the chat or upload a file. Zentra will analyze the visual content to describe it, extract text, or answer specific questions.
- **WebUI Snapshots**: Use the camera icon to take an instant photo via your webcam and send it directly to the AI for real-time analysis.
- **Supported Models**: Vision works with multimodal models (e.g., Gemini, GPT-4o, LLaVA).
