# 👁️ 5. Visión e Interacción Multimodal

Zentra Core soporta nativamente la visión artificial a través de los plugins `VISION_AI` y `WEBCAM`.

- **Análisis de Imágenes**: Arrastra una imagen al chat o sube un archivo. Zentra analizará el contenido visual para describirlo, extraer texto o responder preguntas específicas.
- **Capturas WebUI**: Usa el icono de la cámara para tomar una foto instantánea con tu webcam y enviarla directamente a la IA para su análisis en tiempo real.
- **Modelos Compatibles**: La visión funciona con modelos multimodales (p. ej., Gemini, GPT-4o, LLaVA).
