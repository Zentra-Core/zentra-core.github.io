# 👁️ 5. Visione e Interazione Multimodale

Zentra Core supporta nativamente la visione artificiale tramite i plugin `VISION_AI` e `WEBCAM`.

- **Analisi Immagini**: Trascina un'immagine nella chat o carica un file. Zentra analizzerà il contenuto visivo per descriverlo, estrarre testo o rispondere a domande specifiche.
- **Scatti WebUI**: Usa l'icona della fotocamera per scattare una foto istantanea tramite la tua webcam e inviarla direttamente all'IA per l'analisi in tempo reale.
- **Modelli Supportati**: La visione funziona con modelli multimodali (es. Gemini, GPT-4o, LLaVA).
