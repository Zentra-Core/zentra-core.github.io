# 🔄 6. Response Management

Zentra offers advanced tools to manage the conversation flow.

- **Regeneration**: If you are not satisfied with a response, use the circular arrow button to ask Zentra to try again. The old response will be replaced by a new processing.
- **Interruption**: You can stop response generation or voice synthesis at any time by pressing `ESC` or the stop button in the sidebar.
- **Dynamic Context**: Zentra maintains a memory of the current conversation to ensure coherent and relevant answers.
