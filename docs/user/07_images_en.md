# 🎨 7. Image Generation

Zentra can create original visual content using the `IMAGE_GEN` plugin.

- **How to use it**: Just ask Zentra to "Generate an image of..." or "Draw a...". The AI will understand the request and activate the generation engine.
- **Supported Engines**: By default, it uses **Pollinations.ai** for fast and creative images, but it can be configured for local engines or other cloud services.
- **Chat Integration**: Generated images appear instantly in the chat and can be zoomed or saved to your computer.
