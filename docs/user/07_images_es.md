# 🎨 7. Generación de Imágenes

Zentra puede crear contenido visual original utilizando el plugin `IMAGE_GEN`.

- **Cómo usarlo**: Simplemente pide a Zentra "Genera una imagen de..." o "Dibuja un...". La IA entenderá la solicitud y activará el motor de generación.
- **Motores Soportados**: Por defecto utiliza **Pollinations.ai** para imágenes rápidas y creativas, pero puede configurarse para motores locales u otros servicios en la nube.
- **Integración con el Chat**: Las imágenes generadas aparecen instantáneamente en el chat y pueden ampliarse o guardarse en tu computadora.
