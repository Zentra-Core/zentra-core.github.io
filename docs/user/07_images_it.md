# 🎨 7. Generazione di Immagini

Zentra può creare contenuti visivi originali utilizzando il plugin `IMAGE_GEN`.

- **Come usarlo**: Basta chiedere a Zentra "Genera un'immagine di..." o "Disegna un...". L'IA capirà la richiesta e attiverà il motore di generazione.
- **Motori Supportati**: Di default utilizza **Pollinations.ai** per immagini veloci e creative, ma può essere configurato per motori locali o altri servizi cloud.
- **Integrazione Chat**: Le immagini generate appaiono istantaneamente nella chat e possono essere ingrandite o salvate sul tuo computer.
