# 🛡️ 9. Advanced Security (Zentra PKI)

Zentra puts security at the center, especially when accessing the interface from other devices.

- **Zentra CA**: The system acts as its own Certification Authority (Root CA), generating secure and unique HTTPS certificates for your machine.
- **Protected Connection**: This enables the "green lock" in the browser, allowing the use of protected features like the microphone (WebRTC) across your entire local network (LAN).
- **Mandatory Authentication**: Access to the WebUI always requires a login. You can manage users and passwords from the control panel.
- **Data Isolation (User Vaults)**: Memories, personal files, and avatars are isolated in protected folders (`memory/vaults/username`), preventing cross-access between users.
- **Sandbox**: Code generated by the AI is executed in a protected environment (AST Sandbox) to prevent harmful operations on your operating system.
