# 🛠️ 11. Hardware Troubleshooting

1. **Graphic Interference (Dashboard):** Resolved by joining the UI thread during F7 menu calls.
2. **Logs:** Found in `/logs`. Can be hidden from chat via F7 → SYSTEM.
3. **Audio Trigger Loop:** Adjust the `Energy Threshold` in **F7 → Listening** to calibrate for background noise.
