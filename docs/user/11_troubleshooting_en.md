# 🛠️ 11. Hardware Troubleshooting

1. **Graphic Interference (Dashboard):** Resolved by joining the UI thread during F7 menu calls.
2. **Logs:** Found in `/logs`. Can be hidden from chat via F7 → SYSTEM.
3. **Audio Trigger Loop:** Adjust the `Energy Threshold` in **F7 → Listening** to calibrate for background noise.

### 🆘 Maintenance & Repair Panel
If you experience path issues (e.g. Piper not found) or need to manage the background service:
1. Open the **Zentra Control Panel** (F7 or `/zentra/config/ui`).
2. Go to the **Help** tab.
3. Locate the **🔧 MAINTENANCE & REPAIR** section.
4. From here you can run a **Full System Check**, **Fix Broken Paths**, or **Uninstall the Service**.
