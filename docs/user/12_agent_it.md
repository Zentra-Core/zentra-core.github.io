# 🤖 12. Agente Autónomo y Sandbox

Zentra Core è dotato di un motore di ragionamento avanzato che gli permette di operare come un Agente autonomo.

- **Ciclo Agente**: Zentra può pianificare ed eseguire compiti complessi suddividendoli in sotto-obiettivi, richiamando plugin e strumenti in successione.
- **AST Sandbox**: Ogni volta che l'IA genera codice (Python, algoritmi, logica), questo viene eseguito in una "vivaia" protetta (Sandbox AST). Questo impedisce l'esecuzione di comandi pericolosi per il sistema operativo.
- **Strumenti Automatici**: L'Agente può usare la calcolatrice, gestire file e cercare informazioni senza l'intervento costante dell'utente.
