# 👥 Multi-User & Profile Management
Zentra Core supports multiple independent users, each with their own isolated data and settings.

## Features
- **Isolated Memories**: Conversations and learning context are private to each user.
- **Personal Vaults**: Dedicated storage for files and assets located in `memory/vaults/username`.
- **Custom Avatars**: Personalize your identity within the chat interface.
- **Role-Based Access**: Distinguish between Admin and Guest users.

## Administration
Admins can manage users from the **Users Tab** in the Configuration Panel.
- **Create User**: Add new accounts to the system.
- **Update Profiles**: Change usernames, passwords, and roles.
- **Delete User**: Completely remove a user and their associated vault.
