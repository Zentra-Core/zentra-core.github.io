# 👥 Gestione Multi-Utente e Profili
Zentra Core supporta più utenti indipendenti, ciascuno con i propri dati e impostazioni isolate.

## Caratteristiche
- **Memorie Isolate**: Le conversazioni e il contesto di apprendimento sono privati per ogni utente.
- **Vault Personali**: Spazio di archiviazione dedicato per file e asset situato in `memory/vaults/username`.
- **Avatar Personalizzati**: Personalizza la tua identità nell'interfaccia chat.
- **Accesso Basato sui Ruoli**: Distingue tra utenti Admin e Guest.

## Amministrazione
Gli amministratori possono gestire gli utenti dal tab **Users** nel Pannello di Configurazione.
- **Crea Utente**: Aggiungi nuovi account al sistema.
- **Aggiorna Profili**: Cambia nomi utente, password e ruoli.
- **Elimina Utente**: Rimuove completamente un utente e il suo vault associato.
