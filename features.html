FEATURES

🪐 Core Modules

The system kernel is built on Core Modules. These form the essential foundation for memory management, configuration persistence, and inter-process communication between AI layers.

🧩 Plugin Ecosystem

Logic is modular. Enable or disable specialized plugins for file management, web search, or media control on the fly without rebooting the system core.

🛠️ Extensions

Power-user tools for deep desktop integration. Extensions allow Hecos to interact with external software, system environments, and advanced CLI pipelines.

🌐 MCP Native Bridge

Native Model Context Protocol support. Connect to any MCP Server to instantly grant the AI access to remote databases, APIs, and standardized toolkits.

🤖 Autonomous Agentic Loop

Watch Hecos reason in real-time. By utilizes multi-step Chain-of-Thought, the AI breaks down complex missions, self-corrects, and executes until completion.

🛡️ AST-Security Jail

A sophisticated Abstract Syntax Tree analysis layer filters all AI-generated code. This creates a secure "Jail" where Python can solve analytical problems without compromising the host operating system's integrity.

💾 Semantic Memory

Utilizes SQLite to store not just chat history, but context and learned facts about the user. The AI evolves with you.

🎙️ Voice Interaction

Low latency Speech-to-Text and high-quality local Text-to-Speech (Piper) enable a natural, hands-free experience.

🖥️ System Bridge

Control windows, files, and applications. Hecos can act as your personal OS shell enhanced with AI.
