Drive Editor — Configuration

Integrated code and text editor for Zentra Drive. Supports syntax highlighting for 50+ languages, quick navigation, and live preview.

⚙️ Behavior
Auto-Save

Automatically save files while editing (every 30 seconds).

Confirm before closing with unsaved changes

Warn if attempting to navigate away with pending changes.

Automatic backup before saving

Creates a .bak file in the same folder before overwriting.

🎨 Editor View
Show line numbers

Word Wrap (automatic wrap)

If active, long lines will be visually broken without modifying the file.

Lateral Minimap

Compact preview of the file on the right side of the editor.

🔒 Limits
Larger files will be opened in read-only mode.
Comma-separated list. These files will only open in preview/read-only mode.
ℹ️ Technical Notes
The editor uses Monaco Editor (the core of VS Code).
Syntax highlighting is automatic based on the file extension.
Files are saved via the Drive API (POST /api/drive/save).
