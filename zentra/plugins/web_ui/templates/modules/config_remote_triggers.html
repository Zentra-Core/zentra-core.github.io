📱 Remote Triggers — PTT & Hardware

Enable remote PTT activation via Bluetooth headphones, smartphone media keys, or external HTTP webhooks (Arduino/ESP32).

Allows hijacking headphone buttons. Note: Claims media priority on your device (you might see a "Zentra" notification).
Allows using Volume Up/Down as PTT start/stop when the page is in focus.
Webhook Endpoints
PTT Start: GET /api/remote-triggers/ptt/start
PTT Stop: GET /api/remote-triggers/ptt/stop
PTT Toggle: GET /api/remote-triggers/ptt/toggle
